| -- | -- | -- |

| MAIN4 | 400 Hz | Back right, CW |
| AUX1 | 50 Hz | RC AUX1 |
| AUX2 | 50 Hz | RC AUX2 |
| AUX3 | 50 Hz | RC AUX3 |
