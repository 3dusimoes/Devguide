# Airframe Motor Map

아래 motor map은 어떤 `MAIN` output이 어느 motor로 연결되는지 알려줍니다. 프로펠로의 회전 방향도 표시합니다.

> **Tip** 3-phase motor의 회전 방향은 3개 연결 와이어 중에 2개를 바꾸면 방향이 반대가 됩니다.
